Interactive entertainment reinvented

GrokkSoft is a brand new name in the PC gaming world. This small company, based in Montreal, has the following as its mission statement:

"GrokkSoft inc. develops innovative 3D entertainment packages for the PC and Internet markets"

In the following months, GrokkSoft has a few software titles scheduled. These will range from "The Scepter of Sun" (The first adventure of the Dragon Home series) to a few shoot 'em up games.The company is also developing in parallel its own game authoring package, named "Baseline". More information on Baseline and the other software packages being developed can be found on this site.

Grokksoft is also offering its technical expertise in the form of consulting services. The skill set that can be found in GrokkSoft range from telecommunication, JAVA and Windows programmation to databases.
